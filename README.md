# Student Registration & Management System

This is a **Java Web Application** developed using **JSP, JDBC, MySQL, and Apache Tomcat**.  
The project allows users to **register, login, view student details, edit, and delete records**.

This project is mainly created for **learning JSP, JDBC database connectivity, and basic CRUD operations**.

## âš™ How the Application Works

1. User opens **index.jsp**
2. New users register using **registration.jsp**
3. Registered users login using **login.jsp**
4. After login, user is redirected to **welcome.jsp**
5. User can:
   - View all student records
   - Edit student details
   - Delete student details
6. User can logout using **logout.jsp**
